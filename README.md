# 8086x-lite-assembler
A simulated processor with small memory and registers using a very extremely lite 8086x assembly code
Commands:
1. mov reg constant
2. mov reg reg
3. mov reg address
4. mov address reg
5. add reg []
6. add reg constant
7. add reg reg
8. cmp reg reg
9. jump commands

Program is not finished and has couple of bugs.
